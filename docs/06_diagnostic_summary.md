# 診断サマリと改善レバー

## 一文要約
ダッシュボードは必要なデータを概ね提供しているが、ページ遷移の導線が分かりにくく、1画面で完結しないため目的の情報にたどり着くまで時間が掛かっている。特に売上・粗利・資金・KPI・データ管理で操作と視線誘導が不統一となっている。

## 主要課題 Top3
1. **情報構造の不足**: メニューアイコンとラベルのクリック範囲が狭く、ページ遷移に迷う。
2. **データ一貫性の不足**: 在庫ページが空白のまま、データ管理ウィザードではHTMLタグがそのまま表示されるなど、実装途中が散見。
3. **効率性の不足**: 主要タスクを実施するのに平均30〜60秒を要し、KGI「一目で把握」を達成できていない。CSV/PDF出力ボタンが見当たらないためデータ活用が限定される。

## 改善レバー Top5（効果フェルミ）
- **トップナビゲーションの再設計（クリック領域拡大＋アイコン統一）**: 誤クリック率50％削減、タスク所要時間25％短縮。
- **ダッシュボード構成の再編（KGI直結指標を常時表示し、目的別タブを明確化）**: 指標把握時間を60秒→15秒に短縮。
- **データ管理ウィザードの完成＆CSV出力実装**: データ読み込み準備時間40％削減、業務効率を年9,000円/人向上。
- **在庫ページ機能の実装**: 在庫確認時間20秒→10秒、欠品リスクの先読みを実現。
- **操作ガイドとツールチップ追加**: 迷いポイント50％削減、SUS向上（推定65→80）。

> 対象プロダクト: [kuraiki-jiowsponylvprfxqefkzeo.streamlit.app](https://kuraiki-jiowsponylvprfxqefkzeo.streamlit.app)

## SUS簡易推定とTime-on-Task（フェルミ）
- **SUS簡易推定**: 全体的に情報提供は豊富だが導線や一貫性に課題があり、回答者10名を想定するとスコアは約65/100と推定。主要課題を解決すれば80点台も期待できる。
- **Time-on-Task推定（全体）**: 主要タスク平均40秒、1日5回閲覧、月20営業日とすると約4,000秒/月（約66分）を要する。改善後に平均15秒/タスクに短縮できれば月1,500秒（25分）となり、1人当たり月約40分の時間削減（時給3,000円なら月2,000円程度）＋意思決定スピード向上の副次効果が期待できる。
